**Очищено від:**

### Нова стаття (new-article.html)

**Створено на основі BBC-шаблону з оригінальної статті.**

**Збережено з оригінального шаблону:**
- Структура HTML (header, article-hero, article-body, footer)
- Кольорова палітра BBC (червоний #b80000, темний #1d1d1d)
- Типографіка (serif для тексту, sans-serif для заголовків)
- Розмітка (категорія, автор, дата, цитати, тематичні теги)
- Footer з навігацією

**Новий контент:**
- **Тема:** Next-Gen Consoles 2027 — огляд майбутніх ігрових консолей
- **Структура:** H1 заголовок, H2 підзаголовки (3 секції), H3 під-підзаголовки
- **Елементи:** абзаци, списки, цитати, 3 зображення
- **3 зображення:** всі з Unsplash (геймінг сетап, VR-шолом, esports)

**Зображення та їхнє розташування:**

1. **Featured (hero) image:** Геймінг сетап з монітором — на початку статті, як головне зображення
2. **Inline image 1:** Людина в VR-шоломі — після секції про нейроінтерфейс PS6, логічно пов'язана з темою
3. **Inline image 2:** Competitive esports — у секції про PC gaming, ілюструє конкурентність платформ
